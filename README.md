# Panna-Store-Sales-Analysis
Retail Sales Data Analysis Project

Objective:
Analyze retail sales data to understand product and regional performance and identify improvement areas.

Tools Used:
- Excel (Pivot Tables, Charts, Data Cleaning)
- SQL (Aggregation, GROUP BY)
- AI tools for business insights

Analysis Performed:
- Product-wise revenue and profit analysis
- Region-wise performance analysis
- Identification of top and weak products and regions

Key Insights:
- iPhone is the highest revenue-generating product
- Headphones is the lowest-profit product
- North and South regions perform best
- West region shows weak performance

Outcome:
Generated data-driven business recommendations to improve sales and profitability.
